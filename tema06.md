# Тема 6. Симетрична та асиметрична криптографія

## 1. Основні поняття

**Криптографія** — наука про методи захисту інформації шляхом її перетворення.

| Термін | Визначення |
|--------|------------|
| **Відкритий текст** | Вихідне нешифроване повідомлення |
| **Шифрований текст** | Результат шифрування |
| **Ключ** | Секретний параметр алгоритму шифрування |
| **Шифрування** | Перетворення відкритого тексту на шифрований |
| **Дешифрування** | Зворотнє перетворення |

---

## 2. Потокові та блокові шифри

### Потокові шифри
- Шифрують дані **побітно або побайтово**
- Приклади: Віженер (з авто-ключем), Вернам, RC4
- Швидкі, підходять для потоків даних у реальному часі

### Блокові шифри
- Шифрують фіксовані **блоки** даних (зазвичай 64 або 128 біт)
- Краще вивчені та ширше застосовуються
- Приклади: DES, AES, 3DES

---

## 3. Шифр Файстеля

Основа більшості блокових шифрів (DES та ін.).

**Принцип**: блок ділиться на дві половини (L, R), які по черзі обробляються через функцію раундів.

```
Раунд i:
  L(i) = R(i-1)
  R(i) = L(i-1) XOR F(R(i-1), K(i))
```

### Дифузія та конфузія (Шеннон):
- **Дифузія** — кожен біт відкритого тексту впливає на багато бітів шифрованого (перестановки)
- **Конфузія** — складний зв'язок між ключем і шифрованим текстом (підстановки)

---

## 4. Симетрична криптографія

**Симетричне шифрування** — один і той самий ключ для шифрування та дешифрування.

```
Шифрування:  C = E(K, P)
Дешифрування: P = D(K, C)
```

### DES (Data Encryption Standard)
- Розмір блоку: **64 біти**
- Розмір ключа: **56 біт** (ефективних)
- 16 раундів Файстеля
- Вважається **ненадійним** (атака грубою силою за < 1 дня)

### 3DES (Triple DES)
- Три послідовні операції DES
- Ефективна довжина ключа: 112 або 168 біт
- Повільніший за DES утричі

### AES (Advanced Encryption Standard)
- Переможець конкурсу NIST (2001)
- Розмір блоку: **128 біт**
- Розмір ключа: **128, 192 або 256 біт**
- 10/12/14 раундів залежно від ключа
- **Сучасний стандарт**

### Режими роботи блокових шифрів:
| Режим | Назва | Особливість |
|-------|-------|-------------|
| ECB | Electronic Codebook | Кожен блок шифрується незалежно — небезпечний |
| CBC | Cipher Block Chaining | Кожен блок XOR з попереднім — безпечніший |
| CFB | Cipher Feedback | Перетворює блочний у потоковий |
| OFB | Output Feedback | Генерує ключовий потік |
| CTR | Counter | Паралельне шифрування, лічильник |

---

## 5. Асиметрична криптографія

**Асиметричне шифрування** — два різних ключі: відкритий (public) і закритий/приватний (private).

```
Шифрування:  C = E(KU_B, P)   — ключем одержувача
Дешифрування: P = D(KR_B, C)  — приватним ключем
```

### Основні властивості:
- Знаючи відкритий ключ, неможливо (практично) отримати приватний
- Відкритий ключ розповсюджується вільно
- Приватний ключ зберігається в таємниці

### RSA
- Найпоширеніший асиметричний алгоритм
- Безпека базується на складності факторизації великих чисел
- Розміри ключів: 1024, 2048, 4096 біт
- Повільніший за симетричне шифрування

### Обмін ключами Діффі–Хеллмана
- Перший протокол для безпечного обміну ключами
- Базується на складності дискретного логарифму
- Не шифрує повідомлення — лише встановлює спільний секрет

```
Аліса:  g^a mod p
Боб:    g^b mod p
Спільний секрет: g^(ab) mod p
```

## 6. Порівняння симетричного та асиметричного шифрування

| Критерій | Симетричне | Асиметричне |
|----------|------------|-------------|
| **Ключів** | 1 (спільний) | 2 (відкритий + приватний) |
| **Швидкість** | Швидке | Повільне (у 100–1000 разів) |
| **Управління ключами** | Складне (потрібен безпечний канал) | Просте (відкритий ключ вільно розповсюджується) |
| **Розподіл ключів** | Проблема | Вирішено |
| **Застосування** | Шифрування даних | Обмін ключами, підписи |

### Гібридна схема:
1. Асиметричне шифрування → передача симетричного ключа
2. Симетричне шифрування → шифрування основних даних

---

## 7. Режими підтвердження автентичності повідомлень

**MAC (Message Authentication Code)** — код автентифікації повідомлення на основі спільного ключа.
**HMAC** — MAC на основі хеш-функції.

---

## Ключові терміни

| Термін | Визначення |
|--------|------------|
| AES | Advanced Encryption Standard — симетричний блоковий шифр |
| RSA | Асиметричний алгоритм на основі факторизації |
| Діффі-Хеллман | Протокол обміну ключами |
| Шифр Файстеля | Структура більшості блокових шифрів |
| Дифузія / Конфузія | Принципи Шеннона для стійкого шифрування |
